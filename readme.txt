A plugin to display social share icons.

== Description ==

Icons roll out on hover and allow the visitor the share the current page to their favourite social networks.

== Installation ==

1. Upload the kmo-social directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Select target social networks and modify any options via Settings > KMO Social
